This is an exercice to realise a mockup in nextjs, shadcn-ui, tailwind for Uptoo.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
